# 🎮 Jogos da Memória - Animes

Projeto desenvolvido com HTML, CSS e JavaScript, com diferentes jogos da memória baseados em animes.

🔗 **Acesse o projeto:**
https://jogos-da-memoria-animes-3wt7.vercel.app/

---

## 📌 Sobre o projeto

Este site reúne vários jogos da memória com temas de animes diferentes, cada um com seu próprio estilo visual e personagens.

Os jogos disponíveis incluem:

* 🟢 Spy x Family
* 🟤 Diários de uma Apotecária
* 🔵 Frieren e a Jornada para o Além
* ⚫ Toilet Bound Hanako-kun

Cada jogo possui:

* Cartas embaralhadas automaticamente
* Sistema de verificação de pares
* Feedback de vitória
* Estilo visual personalizado de acordo com o anime

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Funcionalidades

* Virar cartas ao clicar
* Comparação entre pares
* Bloqueio temporário ao verificar cartas
* Mensagem de vitória ao completar o jogo
* Navegação entre diferentes jogos

---

## 🎨 Design

Cada jogo foi estilizado com cores inspiradas no seu respectivo anime:

* Spy x Family → verde e rosa
* Diários de uma Apotecária → tons terrosos
* Frieren → branco e azul
* Hanako-kun → preto e azul escuro
